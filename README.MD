2. How do you create and insert a new element into the DOM?

Prothome ekta new element create korte hoy, tarpor oi element er moddhe text ba attribute add korte hoy, then parent element er moddhe append or insert korte hoy.

3. What is Event Bubbling and how does it work?

Event Bubbling mane holo kono child element e event ghotle seta step by step parent, then tar parent, erokom kore upore upore propagate kore. Mane event niche theke upore jai.

4. What is Event Delegation in JavaScript? Why is it useful

Event Delegation holo ekta parent element e event listener add kora, arpor event.target diye check kora je kon child element theke event ta asche. Eta useful karon ekta parent e ekta listener diye onek gula child ke handle kora jay, memory kom use hoy, ar dynamically create kora child element er jonnoo kaj kore.

5. What is the difference between preventDefault() and stopPropagation() methods

preventDefault()  browser er default action ke stop kore jemon link e click korle notun page e jawa, form submit howe

stopPropagation()  event ke parent element gulo te propagate hote dey na mane bubbling ke stop kore
